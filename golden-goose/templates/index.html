Welcome to Golden Goose!

A complete Flask application skeleton with all the essentials.

Features Included

🚀 Flask Core

Application factory pattern with blueprints

💾 Database

SQLAlchemy ORM with Flask-Migrate

📝 Forms

Flask-WTF for form handling and validation

🎨 Templates

Jinja2 templates with base layout

📡 API Routes

RESTful API endpoints with JSON responses

⚙️ Configuration

Environment-based config with dotenv

Getting Started

  1. Install dependencies: pip install -r requirements.txt
  2. Copy .env.example to .env
  3. Initialize database: flask db init && flask db migrate && flask db upgrade
  4. Run the app: python app.py
