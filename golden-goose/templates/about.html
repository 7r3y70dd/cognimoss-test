About Golden Goose

Golden Goose is a complete Flask application skeleton designed to help you get started quickly with web development in Python.

What's Included

  • Flask Application Factory: Scalable app structure using the factory pattern
  • Database Integration: SQLAlchemy ORM with example models (User, Post)
  • Database Migrations: Flask-Migrate for managing schema changes
  • Form Handling: Flask-WTF with validation examples
  • Blueprint Architecture: Organized route structure
  • Template System: Jinja2 templates with base layout and inheritance
  • Static Assets: CSS and JavaScript file structure
  • Configuration Management: Environment-based config with python-dotenv
  • API Endpoints: RESTful JSON API examples

Technology Stack

  • Flask 3.0.0
  • SQLAlchemy (via Flask-SQLAlchemy)
  • WTForms (via Flask-WTF)
  • Jinja2 (built into Flask)
